Module 1: Love Yourz





About the Piece



The theme of the piece is self-love. Oftentimes, people get caught up comparing their lives to the lives of others.
In doing so, they lose sight of all the blessings that surround them. The words portrayed on the screen,
"No such thing as a life that's better than yourz" are lyrics from a song called "Love Yourz," written by rapper J Cole.
It is also centered around the theme of self-love, especially in the age of technology, where it is so easy to compare
oneself to others.



I made several design choices to best portray self-love. I made every word besides "YOURZ" lowercase to
emphasize the importance of enjoying ones own life. The word "YOURZ" is also generated in a random color,
while all the other words are white in order to show how everyone's life is different and to highlight the
beauty in that difference. There is a delay between each of the words appearing to mimick the rythym of singing
the lyrics, and the lyrics are spread across four displays: one each for the groups "no such thing", "as a thing",
"thats better than", and "YOURZ." I sepearated this text to both make sure that the text was large enough to
easily read and to represent cyclical nature of the four seasons. Similar to the seasons, life will have its
its ups and downs and there's beauty in the resilience to make it through the cycle.



Setup

Creating the exhibit requires a couple of steps. The first is to download the Arduino software, in my case,
I needed to download the Legacy version, an older version of the software. Once it is downloaded and running,
go to file->preferences and then enable the esp32 board by inserting this url where it says additional boards manager:
https://dl.espressif.com/dl/package_esp32_index.json
Then go to Tools->Boards->ESP32 Arduino and select TTGO T1. In order to select a port, make sure that the ESP32
is connected to the your machine using a data transfer USB-C cable. In my case, I needed to download the following driver
and reset my machine in order to get the code to load on my ESP32.
http://www.wch.cn/downloads/CH34XSER_MAC_ZIP.html
For those that also need to download the driver, select the port /dev/cu.wchusbserial.
It is also required to install the TFT_eSPI library by Bodmer. This can be done by going to Tools->Library Manager.
Once all of the prepation is done, you can upload the code found in Love_Yourz.ino to the ESP32. The file can be
found in the following Git Repository:
https://github.com/aot2109/Module1
